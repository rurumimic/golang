## Install

1. Install [gvm](https://github.com/moovweb/gvm)
1. Releases: `gvm install`
1. Install go: `gvm install goX.Y.Z --binary`
1. Go versions: `gvm list`
1. Select: `gvm use goX.Y.Z`
1. Env vars: `go env`
